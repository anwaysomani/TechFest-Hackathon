TechFest '19 Hackathon

Front-End Hackathon

Applicants are required to create an application(Web/Mobile), and the applications will be tested for User Interface and User Experience within the application.

Back-End Hackathon

Applicants are required to create an application(Web/Mobile), and the applications will be tested for data transaction and data processing.

Full-Stack Hackathon

Applicants are required to create an application(Web/Mobile), and the applications will be tested for UI/UX, as well as data transaction & processing within it.

Competitve Challenge

Applicants are provided with a set of problem statements which needs to be resolved in the stipulated time, in any language applicant is comfortable with.
